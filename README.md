# Golden Crust Bakery — Demo Site

A premium artisan bakery demo website built by [Impact Web Company](https://impactwebco.com) to showcase hospitality web design skills.

## About

This is a portfolio piece demonstrating:
- Premium light-mode web design
- Online ordering UI with pickup scheduling
- Smooth GSAP animations and scroll effects
- Signature bread transformation animation (dough → rising → baked)
- Mobile-first responsive design
- Conversion-focused layout and UX

## Tech Stack

- HTML / CSS / JavaScript (no framework, no build tools)
- GSAP 3.12 + ScrollTrigger (CDN)
- Google Fonts (CDN)

## Local Development

Open `index.html` in a browser or use any local server (e.g., Live Server, Antigravity).
