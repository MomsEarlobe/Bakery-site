# Golden Crust Bakery — Project Overview

## What This Is
A premium demo website for an artisan bakery in Sioux Falls, SD. Built by Impact Web Company as a portfolio piece to demonstrate hospitality web design skills and win real bakery, cafe, and restaurant clients.

## Key Facts
- **Business:** Golden Crust Bakery — artisan luxury bakery
- **Location:** Sioux Falls, SD
- **Pricing:** Mid-tier (accessible premium)
- **Ordering:** Online ordering with pickup scheduling (UI-only for demo)
- **Site type:** Single-page scrolling site
- **Tech:** HTML/CSS/JS, GSAP 3.12 + ScrollTrigger (CDN)
- **Theme:** Light mode, warm cream/beige palette, gold accents
- **Tone:** Warm, friendly, clean, slightly upscale

## What Makes This Site Different
- **Bread transformation animation** — the signature interaction (dough → rising → baked bread)
- **Light, warm design** — unlike the dark themes of other IWC demo sites
- **Online ordering UI** — demonstrates e-commerce capability
- **Photo gallery** — showcases visual storytelling
- **Premium but accessible** — not cold luxury, not cheap casual

## Site Sections
1. Hero — headline + bread animation + primary CTA
2. About / Our Story — bakery philosophy and craft
3. Menu / What We Bake — categorized menu display
4. Online Ordering — ordering system UI with pickup scheduling
5. Gallery — photos of baked goods, bakery space, process
6. Testimonials — customer reviews and social proof
7. Email Signup — subscribe for updates and seasonal menus
8. Location & Hours — map, address, hours, contact
9. Footer — social links, quick nav, brand mark

## Content Constraints
- Do not invent specific menu prices, staff names, or supplier details
- Mark anything unconfirmed as [PLACEHOLDER]
- All menu items should feel realistic for a Sioux Falls artisan bakery
- Copy should be warm and natural — never generic or AI-sounding

## Agent System
This project uses a multi-agent workflow:
- **strategist** — plans sections, content priorities, conversion flow
- **copywriter** — writes all copy, headlines, CTAs, menu descriptions
- **ui-designer** — builds HTML/CSS layout and visual design
- **animation-director** — implements motion, bread animation, scroll reveals
- **qa-reviewer** — reviews all work for quality and consistency

## Quality Bar
The site must be polished enough to show real potential clients. It should feel like a real bakery website, not a template or placeholder.
